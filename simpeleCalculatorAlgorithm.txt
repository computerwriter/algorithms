Title: simple calculator algorithm
1 TAKE operator from the user
2 TAKE two inputs from the user
3 IF operator is equal to +
4       display number 1 plus number 2
5 ELSE IF operator is equal to -
6       display number 1 minus number 2
7 ELSE IF operator is equal to /
8       display number 1 divided by number 2
9 ELSE IF operator is equal to %
10       display number 1 medullas by number 2
11 ELSE IF operator is equal to *
12       display number 1 times number 2
13 ELSE
14       display this option or operator is not valid. try again with a new one
15 REPEAT line 1
